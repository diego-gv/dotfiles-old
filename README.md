# diego-gv's dotfiles

## Original project
https://github.com/georgijd/dotfiles

## How to install

```bash
git clone https://github.com/diego-gv/dotfiles.git ~/dotfiles
cd ~/dotfiles
make
```

> For more information run `make help`
