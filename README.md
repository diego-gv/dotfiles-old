# diego-gv's dotfiles

## Compatibility

- Ubuntu 22.04
- Ubuntu 22.04 (WSL)

## How to install

**Requirements**

```
apt install make git
```

**Installation**

```sh
git clone https://github.com/diego-gv/dotfiles.git ~/dotfiles
cd ~/dotfiles
make
```

**Reboot**

```sh
reboot
```

> For more information run `make help`
